# community-events Web Application
Internet Programming (CSYM019-SUN-S2) Assignment
This is dynamic full-stack web application for managing and displaying community events,
built for the Internet Programming module CSYM019 at the University of Northampton.
## How to Run the Project

1. Make sure Docker is installed.
2. Clone the repository:
   git clone https://github.com/kasragerami68/community-events.git
3. Open terminal and run:
   cd community-events
   docker compose up -d
4. Open in browser:
   - http://localhost:9090 (main site)      login admin page : admin   password: admin123
   - http://localhost:9091 (phpMyAdmin)
