# Infrastructure as Code (IaC) Best Practices for Multi-Cloud Environments

## Project Overview
This project demonstrates Infrastructure as Code best practices for managing resources across multiple cloud providers (AWS and Google Cloud Platform).

## Author
Kasra Gerami
Master's Thesis - Computing (Internet Technology and Security)

## Cloud Providers
- Amazon Web Services (AWS)
- Google Cloud Platform (GCP)

## Tools & Technologies
- Terraform - Infrastructure provisioning
- Checkov - Security scanning
- GitHub Actions - CI/CD automation
- Git - Version control

## Project Structure
```
iac-multi-cloud-thesis/
├── aws/              # AWS-specific infrastructure
├── gcp/              # GCP-specific infrastructure
├── modules/          # Reusable Terraform modules
├── policies/         # Security policies and checks
├── .github/          # GitHub Actions workflows
└── README.md         # This file
```
